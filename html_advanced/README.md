# Holberton School - Web Front End

## HTML Advanced

This project covers advanced HTML concepts and semantic structuring of web pages for Techium, a digital agency.

### Learning Objectives
- HTML5 document structure
- Semantic HTML tags (header, main, footer, article, nav, section, aside)
- Headings hierarchy (h1-h6)
- Lists (unordered, ordered, definition)
- Links, images, tables
- Embedding media (video, audio, iframe)
- W3C compliance
