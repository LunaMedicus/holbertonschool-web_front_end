# CSS Advanced

Styling the Techium digital agency homepage using advanced CSS techniques.

## Learning Objectives
- CSS custom properties (variables)
- Font styling with Google Fonts
- Normalize.css for browser consistency
- Box-sizing model
- Grid system with floats
- Dark theme with data attributes
- Transitions and hover effects
- Card-based layouts
- Navigation styling
- Pseudo-elements for decorative content

## Project Structure
- `index.html` - The Techium homepage
- `styles/` - 32 progressive CSS files (1-style.css through 32-style.css)
- `images/` - Image assets (logos, photos, favicon)
